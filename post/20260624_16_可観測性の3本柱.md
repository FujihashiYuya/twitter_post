# 投稿文

「監視＝ログ」だと足りないと痛感した。可観測性はログ・メトリクス・トレースの3本柱。

エラー率やp95はメトリクスで気づき、多段処理のどこで詰まったかはトレースで特定し、詳細はログで見る。

ログ全文をgrepして傾向を追うのは高コストで遅い。最初に3つ揃える前提で組むのが効く。
