# 投稿文

AWSのEC2・ECR・ECSの違いを整理してみた。

EC2 → サーバーそのもの（仮想マシン）
ECR → コンテナイメージの倉庫
ECS → コンテナの運行管理

ECRに保管したイメージをECSで動かす。
その実行先がEC2かFargateかを選ぶ、という関係。
